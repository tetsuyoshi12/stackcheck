# Project Structure

## 開発方針
- AI-DLC（AI駆動型開発サイクル）によるスピード重視の開発
- MVP（v1）を最速でリリースし、サイクルを回すことを最優先とする
- フロントエンドはAIツール（v0 / Bolt.new 等）で自動生成する前提

## v1 スコープ

### 含む機能
- トピック一覧表示・選択
- クイズ実施（1問ずつ出題・即時フィードバック・解説表示）
- セッション結果表示
- 管理者用トピック・問題登録画面

### 含まない機能（v2以降）
- ユーザー認証
- ダッシュボード・スキルマップ・継続グラフ
- カテゴリ機能
- 習熟フラグ管理
- 次トピックサジェスト

## 非機能要件

| 項目 | 方針 |
|---|---|
| 認証・認可 | 管理APIは固定トークンで保護（本格認証はv2） |
| レスポンス速度 | 問題取得はDBから返すのみ。リアルタイムAI生成なし |
| APIコスト | ユーザーセッション中のAPIコール：ゼロ |
| インフラコスト | Render無料枠で運用。v2移行時に再評価 |

## v1 成功指標

| 指標 | 目標 |
|---|---|
| リリース | Renderへのデプロイ完了・公開URL取得 |
| 初期コンテンツ | トピック数：最低10件投入 |
| 動作確認 | クイズ→即時フィードバック→結果表示の一連フローが機能すること |
