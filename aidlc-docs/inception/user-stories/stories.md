# ユーザーストーリー

## ペルソナ: エンジニア（一般ユーザー）

---

### US-01: トピック一覧の閲覧

**As a** エンジニア  
**I want to** 利用可能なトピックの一覧を見たい  
**So that** 自分が確認したい知識領域のトピックを選べる

**受け入れ基準:**  
トップ画面にトピック一覧が表示され、各トピックのタイトルが確認できる。トピックが1件もない場合は適切なメッセージが表示される。

**INVEST チェック:** Independent ✓ / Negotiable ✓ / Valuable ✓ / Estimable ✓ / Small ✓ / Testable ✓

---

### US-02: クイズの実施

**As a** エンジニア  
**I want to** 選んだトピックの問題を1問ずつ解きたい  
**So that** 自分の知識定着度を確認できる

**受け入れ基準:**  
トピックを選択するとクイズが開始され、問題文と4つの選択肢（A/B/C/D）が表示される。選択肢をクリックすると回答が確定し、次のステップに進む。問題番号（例：1/5）が常に表示される。

**INVEST チェック:** Independent ✓ / Negotiable ✓ / Valuable ✓ / Estimable ✓ / Small ✓ / Testable ✓

---

### US-03: 即時フィードバックと解説の確認

**As a** エンジニア  
**I want to** 回答直後に正誤と解説を確認したい  
**So that** 間違えた理由をすぐに理解して学習効果を高められる

**受け入れ基準:**  
回答後、正解・不正解が即座に表示される。正解の選択肢と解説文が表示される。「次へ」ボタンで次の問題に進める。

**INVEST チェック:** Independent ✓ / Negotiable ✓ / Valuable ✓ / Estimable ✓ / Small ✓ / Testable ✓

---

### US-04: セッション結果と振り返り

**As a** エンジニア  
**I want to** 5問終了後に結果と全問の振り返りを見たい  
**So that** 自分の正答率と苦手な問題を把握できる

**受け入れ基準:**  
5問終了後に結果画面が表示され、正解数（例：3/5）が確認できる。全問の問題・自分の回答・正解・解説が一覧で振り返れる。「別のトピックに挑戦」と「もう一度」ボタンが表示される。

**INVEST チェック:** Independent ✓ / Negotiable ✓ / Valuable ✓ / Estimable ✓ / Small ✓ / Testable ✓

---

## ペルソナ: 管理者（開発者）

---

### US-05: トピックの登録（簡略）

**As a** 管理者  
**I want to** ブラウザからトピックを登録したい  
**So that** クイズコンテンツをDBに追加できる

**受け入れ基準:**  
Basic認証でログイン後、トピック名を入力して登録できる。登録成功・失敗のフィードバックが表示される。

**INVEST チェック:** Independent ✓ / Negotiable ✓ / Valuable ✓ / Estimable ✓ / Small ✓ / Testable ✓

---

### US-06: 問題の登録（簡略）

**As a** 管理者  
**I want to** ブラウザから問題・選択肢・正解・解説を登録したい  
**So that** AIツールで生成したコンテンツをDBに投入できる

**受け入れ基準:**  
トピックを選択し、問題文・選択肢A〜D・正解・解説・出題順を入力して登録できる。登録成功・失敗のフィードバックが表示される。

**INVEST チェック:** Independent ✓ / Negotiable ✓ / Valuable ✓ / Estimable ✓ / Small ✓ / Testable ✓
