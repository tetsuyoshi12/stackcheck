# ペルソナ定義

## Persona 1: エンジニア（一般ユーザー）

| 項目 | 内容 |
|---|---|
| 名前 | 田中 翔（仮名） |
| 職種 | ソフトウェアエンジニア（経験2〜5年） |
| 目的 | 自分の現状スキルを把握したい。インプットした内容が定着しているか確認したい。 |
| 課題 | 書籍や記事でインプットはするが、記憶に定着しているか確認する手段がない。自分の弱点分野がわからない。 |
| 行動特性 | スキマ時間（通勤・休憩）にスマホやPCでサクッと確認したい。長時間の学習より短時間の反復を好む。 |
| 技術リテラシー | 高い。Webアプリの操作に慣れている。 |

**このペルソナが使う機能：**
- トピック一覧の閲覧・選択
- クイズの実施（4択回答）
- 即時フィードバック・解説の確認
- セッション結果の振り返り

---

## Persona 2: 管理者（開発者）

| 項目 | 内容 |
|---|---|
| 名前 | 開発者本人 |
| 職種 | このサービスの開発・運営者 |
| 目的 | クイズコンテンツ（トピック・問題・解説）をDBに登録・管理したい。 |
| 課題 | AIツールで生成した問題コンテンツを効率よくDBに投入したい。 |
| 行動特性 | ブラウザからフォームで入力する。curlより使いやすいUIを好む。 |
| 技術リテラシー | 非常に高い。Basic認証の設定・環境変数の管理ができる。 |

**このペルソナが使う機能：**
- 管理者用トピック登録画面
- 管理者用問題登録画面（選択肢・正解・解説を含む）
